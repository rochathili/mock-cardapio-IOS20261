# Cardápio - Mock Up

## Descrição

Este projeto é um **mock up** desenvolvido utilizando o **Google Stitch** para gerar um exemplo visual a ser seguido nas aulas de **Desenvolvimento Web** da turma **08 de 2026.01**.

## Objetivo

O objetivo deste projeto é fornecer um protótipo visual e interativo que sirva como referência para os estudantes durante as aulas práticas de desenvolvimento web, permitindo que compreendam os conceitos de layout, design e boas práticas de interface.

## Ferramentas Utilizadas

- **Google Stitch** - Para criação e prototipagem do mock up visual

## Como Utilizar

Este projeto serve como guia de referência visual. Os alunos devem analisar o design apresentado e implementar as funcionalidades utilizando HTML, CSS e JavaScript (ou conforme orientado pelo instrutor).
